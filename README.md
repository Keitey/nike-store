# E-Commerce Nike

Este projeto mostra um e-commerce fictício da nike onde o usuário pode escolher pelo modelo de tênis, cor, tamanho e abrir um modal para compra do seu tênis nike favorito.

## Veja o projeto completo:

https://keitey.github.io/nike-store/
